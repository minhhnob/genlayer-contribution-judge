# Gen Portal Submission Description

**Project:** Proof-of-Contribution Judge

Proof-of-Contribution Judge is a GenLayer Intelligent Contract demo that reviews contributor campaign submissions using AI reasoning and transparent scoring.

**Why GenLayer:** this use case requires subjective evaluation of contribution evidence, campaign relevance, and quality. Traditional deterministic smart contracts cannot easily judge natural-language work or reason over a rubric. GenLayer enables validators to evaluate the submitted context and return a consensus-backed result.

**What I built:**

- Intelligent Contract for AI-assisted contribution review.
- Frontend demo for entering contributor evidence and previewing outcomes.
- Deployment guide for GenLayer Studio.
- Blog draft explaining the concept and what I learned.
